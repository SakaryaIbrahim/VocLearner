# VocLearner:
-	Eine Webanwendung, in der es möglich sein soll, Vokabeln zu lernen, indem man erstmal „Lernzettel“ erstellt und in diese die Vokabeln (samt Übersetzung) einträgt
-	Es können in einen Lernzettel beliebig viele Vokabeln eingetragen werden, verändert werden oder auch gelöscht werden
-	Vokabeln MÜSSEN immer mit der Übersetzung eingetragen werden
-	Beim „Starten“ der eigentlichen Anwendung wird einem dann die Vokabel angezeigt und man muss darunter die Übersetzung eingeben
-	Wenn der Nutzer die Übersetzung korrekt eingegeben hat, dann wird dieser Vokabel seltener vorkommen als andere, wenn falsch, dann öfter -> Stufensystem
-	Es gibt keine echte Grammatik und auch keine ausgewählten Sprachen -> Das was der Nutzer in den Lernzettel eingetragen hat ist richtig
-	Das Ganze soll so einfach und selbsterklärend wie möglich gehalten werden -> keine großartigen Startseiten mit Tonnen von Text etc.
-	Eine Benutzeranmeldung wäre schon sehr nützlich, damit jeder seine eigenen Individuellen Lernzettel hat, falls wir das aber nicht schaffen, dann wäre eine Option es so zu machen, dass jeder Lernzettel eindeutige Namen haben muss und man diese über eine Suchfunktion finden kann -> Folge wäre jedoch, dass Lernzettel von ausnahmslos jeden Nutzer eingesehen, genutzt und verändert werden können -> Thema Sicherheit
